# Nuxt Component DPE

## How to use :

### 1. Install package
```
npm install nuxt-component-dpe
```
### 2. Init component in Nuxt with values

- With Export Mode (PNG):
```
<dpe valueDPE="0" :style="'width:25%;'" exportMode="active"/>
```

- Without Export Mode (PNG):
```
<dpe valueDPE="0" :style="'width:25%;'" exportMode=""/>
```
